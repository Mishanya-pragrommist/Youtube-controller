# Youtube Controller Chrome Extention
Are you tired of doom scrolling YouTube for hours? 
Do you wish to disable YouTube shorts and recommendations 
in order to focus on tutorials/guides or other 
actually useful content and work productevily? 
Then YouTube Controller extention is a solution for you!

## Capabilities of YouTube Controller
Basically, it allows you to block content feed and
recommendations so you use only manual search.
Also you can optionally set timer to work

## How to work with it
1. Install the extention (obviously).
2. Open it (even more obviously).
3. Set your preferencies: what to block, how many 
videous can be shown in Recommendations, how long
would you like to work. 
4. Press "Start" and enjoy absence of useless content.

Also you can always stop the extention by pressing "Stop" button,
or disable it at all.
